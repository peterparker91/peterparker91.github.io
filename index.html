Content Slab

The content slab is the key content component on the site's high-level pages. It is organized in 3-column layouts, that can be leveraged and merged based on the content types to display. The content slab provides the option to display an additional sub navigation to create more depth for similar content categories to display. In many cases the content slab is used dedicating one column to a side module that displays, related aspects of content or interactivity (e.g., entry points to interactive tools).

Usage:

  • Landing Template
  • Product Category Template
  • Article Category Template
  • Your DBS Template
  • iBanking Dashboard

Category Grid

The Category Grid component organizes the content items of that category landing page into a grid. To prime the visitor for the different types of content displayed it displays an additional intro text. Note that this component usually sits below a category page subnavigation.

Similar to the basic category grid, the Collapsible Category Grid component allows to display larger amounts of content with collapsable category headers.

Usage:

  • Product Category Template
  • Article Category Template

Branch Finder

This component allows visitors to find the physical touch-points DBS is providing, such as branches and ATMs. The map component should default to the customers indicative current location (e.g., based on IP segment). Upon entering a postal code, both the map and the results list should refresh.

Usage:

  • Splitter Template
  • Forms Template

Animations

The site uses animations quite sparingly, in order to ensure cross-browser compatibility as well as minimize load times for low bandwidth customers, and there are only a few places where subtle animations have been added to emphasize certain interactions.

Examples:

  • The Splitter Page uses a simple fade to animate between default and hover states.
  • Hero Module uses a combination of fade of the image and right-to-left motion + fade for the text.
